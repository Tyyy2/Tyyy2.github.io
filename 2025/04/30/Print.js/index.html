一款好用的JS列印套件

支援列印 PDF、HTML、圖片、JSON 格式。
使用方式簡單、快速。

IE 不支援 PDF、IMG

安裝
參數
  • printable
  • type
  • header
  • headerStyle
  • maxWidth

用處 : 要列印的物件(PDF、img url、JSON、HTML)
default : null

用處 : 列印物件(printable)類型
default : 'pdf'

用處 : 自訂 HTML/JSON/IMG 列印時的 header，接受文字或 HTML
default : null

用處 : 自訂 HTML/JSON/IMG 列印時的 header 樣式
default : 'font-weight:300;'

用處 : 限制最大寬度
default : 800

  • gridHeaderStyle
  • gridStyle
  • repeatTableHeader
  • repeatTableHeader

用處 : 自訂 JSON 列印時的資料格 header 樣式
default : 'font-weight:bold;'

用處 : 自訂 JSON 列印時的資料格樣式
default : 'border: 1px solid lightgray; margin-bottom: -1px;'

用處 : 是否在每頁顯示 header
default : true

用處 : 是否在每頁顯示 header
default : true

使用
  • PDF
  • HTML
  • image
  • JSON

SqlKata 是什麼?

An elegant Query Builder and Executor helps you deal with SQL queries in an elegant and predictable way. >Written in C# > https://sqlkata.com/docs/

一個專為 .NET 平台設計的輕量級 SQL Builder，透過直覺的 Fluent API，可以用 C# 語法動態組裝 SELECT、INSERT、UPDATE、DELETE 等各式查詢，並支援多種資料庫。
它本身只負責「產生」符合不同 SQL 方言的查詢字串，再搭配例如 Dapper 或純 ADO.NET 去「執行」並映射結果，既能保留手寫 SQL 的靈活性，也能避免大量字串拼接的錯誤，還可有效防止 Sql Injection。
借助 SqlKata，可更專注在查詢邏輯而非字串組合，同時享有良好的可讀性與可維護性。
